Installation
I
Avoir Dies Irae installé
Version Steam recommandée. Langue de base : anglais ou japonais.
II
Localiser le dossier du jeu
Sur Steam : clic droit sur Dies Irae → Propriétés → Fichiers locaux → Parcourir.
Le dossier contient data4.dat, DiesIrae.exe, etc.
III
Copier data5.dat
Placer data5.dat dans le dossier du jeu, au même niveau que data4.dat.
Le moteur Malie charge automatiquement data5 en priorité.
IV
Lancer le jeu
Le texte français apparaît directement. Aucun autre fichier à modifier.

FAQ
Le jeu crash après installation du patch

Vérifier que data5.dat est bien dans le dossier racine du jeu (là où se trouve DiesIrae.exe). Le fichier ne doit pas être dans un sous-dossier.

Certains textes sont encore en anglais

La traduction est en cours. Les passages non traduits affichent le texte anglais d'origine. Revenir régulièrement pour les mises à jour.

Compatible avec les sauvegardes existantes ?

Oui. Le patch ne modifie que le script texte, pas la logique du jeu. Les sauvegardes sont entièrement compatibles.

Version Steam vs version JP ?

Le patch est basé sur la version anglaise Steam. Fonctionne sur la version Steam EN. Compatibilité version JP non garantie.

Comment contribuer à la traduction ?

Contacter l'équipe via le Discord ou ouvrir une issue sur le dépôt GitHub. Des chunks de ~1 000 lignes sont disponibles à l'attribution.
